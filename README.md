# Every Drop Counts 💧

An interactive awareness website developed for the National Service Scheme (NSS), Cambridge Institute of Technology, Bangalore, to educate people about Bengaluru's growing water crisis and encourage responsible water conservation.

## 🌍 About the Project

Bengaluru was once known as the city of lakes. Rapid urbanization, groundwater depletion, and increasing water demand have pushed the city toward a severe water crisis.

**Every Drop Counts** aims to:

* Raise awareness about Bengaluru's water challenges
* Bust common myths surrounding water usage
* Showcase lake restoration success stories
* Help users calculate their personal water footprint
* Encourage citizens to take a public water conservation pledge
* Collect suggestions and community feedback

---

## ✨ Features

### 📊 Water Crisis Dashboard

* Groundwater depletion statistics
* Borewell and lake data
* Future water crisis projections

### 🏞️ Interactive Lake Explorer

* Information about major Bengaluru lakes
* Restoration success stories
* Environmental impact highlights

### 🧮 Water Footprint Calculator

* Estimate daily water consumption
* Personalized efficiency grading
* Usage breakdown by activity

### 🤝 Community Participation

* Water conservation pledge system
* Public suggestion submission
* Contact and volunteer form

### ☁️ Supabase Integration

* Real-time pledge collection
* Suggestion storage
* Contact message management

## 🚀 Deployment

The website is deployed using GitHub Pages.

1. Clone the repository
2. Configure Supabase credentials
3. Create the required database tables
4. Enable Row Level Security (RLS)
5. Deploy via GitHub Pages
